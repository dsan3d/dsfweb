Listado de algunas Apps que Tengo

Cronometro

Esta aplicación es un simple cronómetro con una ventana circular. Con .NET Framework se pueden hacer ventanas con formas geométricas diferentes a las cuadradas. La app está hecha con V. Studio 2008 y VB .NET.
Pulsa aquí para descargarla desde GitHub.

Garajes Sofer CRM

Esta aplicación es un CRUD para gestión de clientes de un negocio de Parking.
Lo hice por encargo para una pyme de Gijón, el 11 de Septiembre de 2001. Estaba hecho originalmente con VB 6, pero esta versión la hice con V. Studio 2002 y VB .NET con .Net Framework 1.
Pulsa aquí para descargarla desde GitHub.

Reproductor MP3

Un reproductor Mp3 para Windows XP.
Esta hecho con Visual Studio 2005, .NET Framework 2 y VB .NET. Utiliza una dll externa de Windows Media player.
Puedes descargar la aplicación desde este enlace de GitHub.

Juego de la Peseta

La primera versión de esta aplicación la hice en QBasic para MSDOS a mediados de los 90. Esta versión es una migración que hice para Windows XP con V. Studio 2002 y VB NET. El código fuente lo perdí (o no recuerdo donde lo guardé).
En el año 2002 publiqué esta app en Softonic. Puedes visitar mi cuenta de github para descargarlo si quieres.

Cryptocoin Calculadora Cryptomonedas

Esta aplicación es un conversor / calculadora de Cryptomonedas y también un lector de noticias. Está publicada en la Windows Store y en Google Play Store. Puedes descargarla desde mi perfil de GitHub.
Esta realizada con Visual Studio 2022 y C# con .NET MAUI. y hace uso de una API REST.
